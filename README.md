# LSB-PYTHON-SCRIPT

给一个朋友的毕业论文写的演示程序，其原理就是利用BMP最低有效位替换为被隐藏内容，由于其在显示层面的加权很低，因此从肉眼几乎无从分辨，但是其实已经加进去了我们特殊的数据。在程序里写入了一个特定的标记，利用这个标记来标识被哪部分是被隐藏的数据。

程序有两个版本，一个是命令行下的，另一个是GUI的。全部由PYTHON写成。依赖项均为PYTHON自带，因此移植比较方便。

## 功能分为四个部分：

1. 文字LSB隐藏
2. 文件LSB隐藏
3. 隐藏文字提取
4. 隐藏文件提取
